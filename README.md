# Photography Work Portfolio

This repository powers Jerry's photography portfolio. It is a Next.js site built with TypeScript, MDX content, and Once UI components, with sections for home, about, work, blog, and gallery.

The content is organized around real portfolio material.

## Content Structure

- Site configuration and global content live in [src/resources/content.tsx](src/resources/content.tsx).
- Theme, routing, and metadata settings live in [src/resources/once-ui.config.ts](src/resources/once-ui.config.ts).
- App routes live under [src/app](src/app).
- Blog posts live in [src/app/blog/posts](src/app/blog/posts).
- Work projects live in [src/app/work/projects](src/app/work/projects).
- Public images and OG assets live in [public/images](public/images).
